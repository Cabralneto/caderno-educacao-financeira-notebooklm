# 📘 Caderno Temático – Educação Financeira com NotebookLM

Projeto acadêmico desenvolvido com foco em aprendizagem ativa utilizando Inteligência Artificial como ferramenta de organização e análise de conhecimento.

---

## 🎯 Objetivo do Projeto

Construir um miniguia estruturado de Educação Financeira Introdutória a partir de fontes abertas oficiais, utilizando o NotebookLM para:

- Organizar conteúdos
- Sintetizar conceitos
- Criar glossário
- Formular perguntas estratégicas
- Testar variações de prompts
- Desenvolver pensamento crítico sobre o uso de IA no estudo

---

## 🧠 Metodologia

O projeto foi desenvolvido em etapas:

1. Curadoria de fontes institucionais confiáveis
2. Importação dos materiais no NotebookLM
3. Definição de objetivos de estudo
4. Formulação de prompts estratégicos
5. Teste de variações de abordagem (didática, técnica e prática)
6. Organização estruturada do conteúdo final

A proposta não foi apenas gerar resumos, mas utilizar a IA como instrumento de análise, comparação e organização do conhecimento.
