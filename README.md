# 📘 Caderno Temático – Educação Financeira

Projeto desenvolvido utilizando o NotebookLM como ferramenta de aprendizagem ativa.

## 🎯 Objetivo
Organizar conteúdos introdutórios de educação financeira a partir de fontes abertas, criando um miniguia estruturado com glossário e prompts reutilizáveis.

## 🧠 Metodologia
- Curadoria de fontes institucionais
- Formulação de perguntas estratégicas
- Teste de variações de prompts
- Organização estruturada do conhecimento

## 📚 Estrutura do Projeto
- Trabalho acadêmico completo
- Glossário de conceitos
- Lista de prompts reutilizáveis
- Referências utilizadas

## 🤖 Ferramenta Utilizada
NotebookLM (Google)
